# EurorackVCO
Clone of Befaco 'even' VCO, original files (accessed 2025-12-12):
https://befaco.org/docs/Even_VCO/
The original pcb states that the design is public domain.

This projects goal is to remake the 'Even VCO' in a 8hp format, to be ordered on a PCBA service like JLCPCB
Files are (for now) easyeda pro.
